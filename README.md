# python_stock_related

written in python 3.9.6 and windows 10

talib link: https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib

#create venv
```bash
py -m venv venv
```

#use venv
```bash
venv\Scripts\activate.bat
```

#install lib
```bash
py -m pip install -r requirements.txt
```

#run program
```bash
py -m main.py
```
